## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

    NodeJS
    Npm

### Setup

1. git clone https://github.com/Kumilachew-g/react-todo-app.git
2. cd Math-Magicians

### Install

    npm install

### Usage

    npm  start
